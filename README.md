## About

林逍遥AI 是一个面向中文用户的 AI 教育品牌，致力于用最易懂的方式帮助每个人掌握 AI 工具，把 AI 融入日常工作与生活。

## Courses

### Claude for Everyone

面向所有人的 Claude AI 入门课，零编程基础即可学习。

- 认识 Claude — 了解它是什么、能做什么
- 学会提问 — 掌握 Prompt 的核心结构
- AI 写作 — 用 Claude 写邮件、报告、改稿
- AI 思考 — 用 Claude 整理思路、分析文档
- AI 工作流 — 把 Claude 融入日常工作
- 进阶技巧 — 多轮对话、验证输出、高效协作
