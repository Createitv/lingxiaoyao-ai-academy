# 角色提示（Role Prompting）

角色提示是最简单、最有效的 Prompt 技巧之一。通过指定 Claude 扮演某个专业角色，可以显著提升回答质量。

## 基础用法

```
你是一位 [角色]，[补充背景]。
请帮我 [具体任务]。
```

## 实战示例

### 法律顾问
```
你是一位熟悉中国劳动法的法律顾问。
我的公司想让员工签署竞业限制协议，请告诉我需要注意哪些法律要点？
```

### 产品经理
```
你是一位有10年经验的B端产品经理。
我在规划一个企业内部知识库系统，请帮我列出核心功能清单，按优先级排序。
```

### 写作编辑
```
你是一位专业的中文写作编辑，擅长商务文章。
请帮我优化以下段落，让它更简洁有力：[粘贴原文]
```

## 注意事项

- 角色设定后，Claude 会一直保持这个角色，直到你重置对话
- 角色越具体，效果越好（"资深Java工程师" 比 "程序员" 效果好）
- 可以结合背景信息：`你是一位擅长...的...，你的受众是...`
