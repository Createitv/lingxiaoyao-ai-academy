# 第十天：激发创意的 AI 伙伴

你有没有遇到过这种情况：需要想一个好点子，但脑子一片空白？或者有了一堆想法，不知道哪个最靠谱？Claude 是你最理想的头脑风暴伙伴——它不会疲倦、不会评判你、能从无数个角度给你灵感。

## 头脑风暴的三个阶段

有效的头脑风暴不是漫无目的地乱想，而是一个有节奏的过程：先发散、再收敛、最后评估。Claude 在每个阶段都能帮上大忙。

## 第一阶段：发散——让想法飞一会儿

发散阶段的关键是数量而非质量。你可以用这些提示词让 Claude 帮你打开思路：

**广度发散**：「请为[目标]生成20个不同方向的创意点子，越多样越好，不需要考虑可行性。」

**角度切换**：「分别从用户、投资人、竞争对手、技术团队四个角度来思考这个问题。」

**类比迁移**：「有哪些其他行业的成功案例可以借鉴到我们的情况中？」

**极端思考**：「如果预算无限，你会怎么做？如果只有100元，你又会怎么做？」

Claude 特别擅长角度切换和类比迁移，它的知识面足够广，能找到你想不到的关联。

## 第二阶段：收敛——整理与筛选

当你有了一堆想法之后，让 Claude 帮你整理：

```
请把刚才生成的所有创意按以下维度分类整理：
1. 按主题方向分组
2. 标注每个想法的创新程度（高/中/低）
3. 标注实施难度（高/中/低）
4. 去除明显重复的想法
5. 从每组���选出最有潜力的1-2个
```

Claude 的分类整理能力非常强，它能迅速找到不同想法之间的关联，帮你理清思路。

## 第三阶段：评估——用矩阵做决策

这是头脑风暴最关键的一步。你可以让 Claude 用评估矩阵来帮你做理性的选择。

```
请用以下标准给筛选出的5个方案打分（1-10分）：
- 市场潜力
- 实施难度（分数越高表示越容易）
- 投入成本
- 竞争优势
- 与现有业务的协同性

最后计算加权总分，并给出你的推荐排序。
```

## SCAMPER 方法：系统化创新

SCAMPER 是一个经典的创新思维工具，和 Claude 配合使用效果极佳。你只需要告诉 Claude：

```
请用SCAMPER方法帮我分析[产品/服务/方案]：
- Substitute（替代）：有什么可以替换？
- Combine（组合）：能和什么结合？
- Adapt（调整）：可以如何改造？
- Modify（修改）：放大或缩小某个特点？
- Put to other uses（其他用途）：还能用在哪？
- Eliminate（消除）：去掉什么会更好？
- Reverse（反转）：反过来做会怎样？
```

## 实战案例：为新产品起名

假设你要为一款 AI 写作工具起名字：

**第一步（发散）**：「请为一款面向自媒体创作者的 AI 写作工具生成30个品牌名候选，风格可以包括中文、英文、中英混合。」

**第二步（收敛）**：「请把这30个名字按风格分类，从每类中选出最好的2个。」

**第三步（评估）**：「请从记忆度、品牌感、域名可用性、目标用户契合度四个维度给每个候选名打分。」

三步下来，你就能得到一个有理有据的品牌名决策。

## 提升创意质量的小技巧

1. **先给 Claude 足够的背景**：你对问题的了解越多，Claude 给出的创意越靠谱
2. **鼓励 Claude 大胆想**：加一句「不需要考虑可行性，先求新求异」
3. **多轮迭代**：觉得某个方向有意思，可以让 Claude 继续深入
4. **引入约束**：「如果只能用10个字来表达这个想法，你会怎么说？」
5. **让 Claude 扮演反对者**：「请对方案A提出最尖锐的3个质疑」

## 今日小结

Claude 作为头脑风暴伙伴的最大优势是：**它能同时从多个角度思考，不会因为惯性思维而局限**。发散时给它自由，收敛时给它标准，评估时给它维度——你就能把 Claude 的创意能力发挥到极致。
