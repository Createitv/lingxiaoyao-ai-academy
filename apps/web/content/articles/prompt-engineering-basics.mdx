# Prompt 工程入门

很多人觉得 AI「不好用」，往往不是 AI 的问题，而是提问方式的问题。

## Prompt 的4个核心要素

一个好的 Prompt，包含以下4个部分：

### 1. 角色（Role）

告诉 AI 扮演什么角色，它会更聚焦。

```
❌ "帮我写封邮件"
✅ "你是一位专业的职场写作顾问，帮我写一封..."
```

### 2. 任务（Task）

明确说清楚你想要什么结果。

```
❌ "写一篇关于AI的文章"
✅ "写一篇800字的科普文章，介绍AI对普通人工作的影响"
```

### 3. 背景（Context）

提供足够的上下文，AI 才能给出准确答案。

```
❌ "帮我分析这份合同"
✅ "这是一份软件服务合同，甲方是我们公司，请重点关注服务期限和违约条款"
```

### 4. 格式（Format）

告诉 AI 你想要什么格式的输出。

```
❌ "总结一下这篇文章"
✅ "用3个要点总结这篇文章，每个要点一句话"
```

## 动手练习

把下面的模糊 Prompt 改写成包含4个要素的好 Prompt：

> "帮我想想怎么跟老板谈加薪"

（参考答案：「你是一位职场谈判专家。我工作3年，表现优秀，希望申请20%加薪。请帮我设计一个谈话框架，包含：开场白、核心论据、可能的反对意见及应对方案。用列表格式输出。」）
