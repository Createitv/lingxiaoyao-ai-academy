## 策略一：角色锁定

角色锁定不只是在开头说"你是一个XXX"，而是在整个对话中持续强化角色设定。当你发现 Claude 回到了通用 AI 助手的口吻，就需要重新锁定。

**示例对话流程**：

```
第1轮：你是一位有15年经验的B端产品经理，擅长SaaS领域。
       请用你的专业视角分析这个需求文档...

第3轮：（当Claude开始用过于学术化的语言时）
       回到产品经理的视角。产品经理不会这么说话，
       请用更实际、更落地的方式重新分析。

第5轮：基于你作为资深PM的判断，这个功能该不该做？
       给我一个明确的结论和三条理由。
```

关键技巧：角色锁定要**具体**（年限、领域、风格），发现偏离时要**及时纠正**。

## 苏格拉底法：用提问引导思考

苏格拉底法的核心是**不直接告诉 Claude 答案，而是通过一系列提问让它自己发现问题**。这种方法在以下场景特别有效：

- 你想验证 Claude 的逻辑是否严密时
- 你希望 Claude 从多个角度审视自己的回答时
- 你在学习某个领域，想通过对话加深理解时

**示例**：

```
Claude 给出了一个市场策略后，你可以问：
"如果竞争对手也采用同样的策略，你的建议还成立吗？"
"这个策略的最大风险是什么？如果失败了怎么办？"
"有没有更简单的方式达到同样的效果？"
```

每一个问题都会迫使 Claude 重新审视自己的回答，往往能得到更深入、更全面的分析。

## 综合实战练习

试试把四种策略组合使用：

1. **开场**用角色锁定，设定一个行业专家
2. **第一轮**用渐进细化，先广泛探索再聚焦
3. **中间**用对比验证，让 Claude 生成并评估多个方案
4. **遇到可疑论点时**用苏格拉底法追问验证
5. **发现跑偏时**用纠偏技巧及时拉回正轨

这四种策略不是互斥的，而是可以灵活组合的工具箱。掌握了这些，你就能在任何复杂场景中驾驭 Claude，真正做到"指哪打哪"。实际使用中，你会逐渐形成自己的对话风格和节奏，这就是从"技巧"到"内功"的转变。
