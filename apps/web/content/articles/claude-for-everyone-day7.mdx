## 6大核心技能速览

### Day 1：认识 Claude
- Claude 是 Anthropic 开发的 AI 助手
- 四大核心能力：写作、分析、思考、对话
- 和搜索引擎的区别：搜索引擎帮你找信息，Claude 帮你处理信息

### Day 2：RTCF 提问法
- **R**ole（角色）：让 Claude 扮演专家
- **T**ask（任务）：明确、具体地描述需求
- **C**ontext（背景）：提供必要的情境信息
- **F**ormat（格式）：指定输出的结构和长度

### Day 3：多轮对话
- 好的结果是「聊出来的」，不要期望一次到位
- 四种策略：渐进式细化、反馈式修正、角度切换、追问深入
- 知道何时开新对话

### Day 4：文档分析
- 支持 PDF、Word、Excel、代码、图片等多种格式
- 三步流程：上传 → 精准提问 → 获取洞察
- 先概述再深入，要求引用原文

### Day 5：AI 辅助写作
- 黄金流程：构思 → 初稿 → 修改 → 定稿
- 邮件、报告、文章、总结四大场景
- 最后10分的个人风格只有你能加上去

### Day 6：翻译与多语言
- Claude 翻译理解语境，不是词对词替换
- 翻译四步法：提供原文 → 说明语境 → 翻译输出 → 审校优化
- 支持50+语言，擅长文化适配

## 实战测验：5道挑战题

现在来检验一下你的 Claude 技能。请打开 Claude，逐一完成以下5道挑战题：

### 挑战1：综合提问（考察 Day 1 + Day 2）

用 RTCF 模型让 Claude 帮你做一件真实的工作任务。要求：4个要素都用上，一次性得到满意的回答。

### 挑战2：多轮优化（考察 Day 3）

让 Claude 帮你写一段自我介绍。要求：至少进行4轮对话，每轮给出具体的修改反馈，最终得到一段你真正满意的文字。

### 挑战3：文档分析（考察 Day 4）

找一份真实的文件（PDF、Word 或截图），上传给 Claude，完成以下任务：
1. 让 Claude 概述文件内容
2. 针对你最关心的部分深入提问
3. 让 Claude 从不同角度分析这份文件

### 挑战4：写作任务（考察 Day 5）

完成一个完整的写作流程：
1. 先让 Claude 帮你列大纲
2. 选择一个段落让 Claude 展开
3. 审阅后给出修改意见
4. 加入你的个人风格做最后调整

### 挑战5：翻译任务（考察 Day 6）

找一段英文内容（邮件、文章或产品说明），让 Claude 翻译成中文。要求：
1. 提供完整的语境信息
2. 指定翻译风格和目标读者
3. 对翻译结果进行至少一轮反馈优化

## 自评打分标准

每道挑战题10分，满分50分：

- **9-10分**：完美完成，结果令人满意
- **7-8分**：基本完成，还有提升空间
- **5-6分**：勉强完成，关键步骤有遗漏
- **5分以下**：需要回顾对应章节重新学习

**40分以上**：你已经具备了扎实的 Claude 基础能力，下周的进阶内容对你来说会很顺利。

**30-39分**：基础不错，建议重点复习得分较低的技能，多做练习。

**30分以下**：别灰心！回顾 Day 1-6 的内容，重点完成每天的实战练习。熟能生巧，多用几次就会有质的飞跃。
