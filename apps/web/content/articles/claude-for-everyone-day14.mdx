### 任务一：市场调研（Day 8 + Day 13）

使用结构化 Prompt 让 Claude 帮你分析大学生学习管理市场：

```
<role>你是一位教育科技行业的市场分析师</role>
<context>我们计划开发一款面向中国大学生的学习管理App</context>
<task>
1. 分析中国大学生群体的学习痛点（至少5个）
2. 评估学习管理类App的市场规模
3. 描述目标用户画像（年龄、行为特征、付费意愿）
4. 列出可能的盈利模式
</task>
<output_format>每个部分用二级标题分隔，数据用表格呈现</output_format>
```

### 任务二：竞品分析（Day 9 + Day 12）

```
请分析以下5款竞品App的核心数据，并以Markdown表格输出：
Notion、Forest、番茄Todo、知犀思维导图、幕布

表格列：App名称、核心功能、月活估计、付费模式、主要优势、主要不足
```

### 任务三：功能规划（Day 10）

用头脑风暴三阶段法：先让 Claude 发散生成20个功能创意，再收敛筛选，最后用评估矩阵选出 MVP（最小可行产品）的核心功能。

### 任务四：原型脚本（Day 11）

```
请帮我写一个简单的HTML原型页面，展示这个学习管理App的主界面。
包含：今日课表、学习计时器、任务清单三个模块。
要求：移动端优先，使用现代化的UI风格。
```

### 任务五：商业计划书（Day 8 + Day 12）

```
<role>你是一位有丰富经验的创业导师</role>
<task>根据以上市场调研和竞品分析结果，撰写一份商业计划书大纲</task>
<output_format>
Markdown格式，包含以下章节：
1. 执行摘要
2. 市场分析
3. 产品方案
4. 商业模式
5. 竞争策略
6. 团队介绍（预留）
7. 财务预测
8. 融资需求
</output_format>
```
