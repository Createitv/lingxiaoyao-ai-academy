# 第一天：认识 Claude

欢迎来到「30天学Claude」系列！

这个系列的目标很简单：让你在30天内，真正把Claude用起来，变成你日常工作的得力助手。

## Claude 是什么？

Claude 是 Anthropic 公司开发的 AI 助手。和你可能听说过的 ChatGPT 类似，Claude 擅长：

- **写作**：邮件、报告、文章、总结
- **分析**：读文件、提炼观点、比较方案
- **思考**：头脑风暴、整理思路、分解问题
- **对话**：解答问题、辅导学习、陪你练习

## 今天的任务

打开 [claude.ai](https://claude.ai)，注册账号（免费版足够入门），然后输入下面这句话：

```
你好！请用3句话告诉我，你最擅长帮人做什么事？
```

看看 Claude 怎么回答。
