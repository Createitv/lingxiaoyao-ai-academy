# 第十六天：深度思考模式——让Claude帮你做复杂决策

你有没有遇到过这样的情况：问 Claude 一个复杂问题，它秒回了一个答案，但你总觉得分析不够深入？今天我们来学习一个强大的功能——**Extended Thinking（深度思考模式）**，让 Claude 先想清楚，再给你答案。

## 什么是深度思考模式？

在普通模式下，Claude 会直接生成回答。而在深度思考模式中，Claude 会先进行一轮内部推理——就像人类在做重要决定前会先在脑中权衡利弊一样——然后再给出最终答案。

关键区别在于中间那一步"内部推理"。在这个过程中，Claude 会像人一样列出不同角度、权衡利弊、验证逻辑，最终给出经过深思熟虑的答案。

## 什么场景适合用深度思考？

深度思考模式不是万能的。简单的问题用普通模式更高效，复杂的问题才值得让 Claude 深度推理。

## 如何在 Claude.ai 中使用？

使用方法非常简单。在 Claude.ai 的输入框中，你可以看到模型选择器。选择支持 Extended Thinking 的模型后，Claude 会自动在需要时启用深度思考。

你也可以在提示词中明确要求 Claude 进行深度思考：

```
请仔细思考以下决策问题，列出所有需要考虑的因素，然后给出你的建议：

我是一个3人创业团队，正在考虑是自建技术团队还是外包开发。
预算50万，产品是一个B2B SaaS工具，需要在6个月内上线MVP。
```

这样的复杂决策问题，深度思考模式会考虑团队规模、预算限制、时间约束、技术复杂度、长期维护等多个维度，给出远比普通模式更全面的分析。

## 普通模式 vs 深度思考的实际对比

**问题**：一家餐厅日均客流量 200 人，客单价 80 元，想投入 20 万翻新装修，预期提升客单价 15%，多久能回本？

**普通模式**的回答可能直接计算：200 x 80 x 15% = 2400 元/天，200000 / 2400 ≈ 83 天。

**深度思考模式**的回答会考虑更多因素：装修期间的营业损失、翻新后初期的客流波动、季节性因素、客流量是否也会变化、折旧成本等，最终给出一个更贴近实际的分析，而不仅仅是简单的数学除法。

## 使用建议

1. **不要所有问题都开深度思考**——简单问题用普通模式更快更省资源
2. **给足背景信息**——思考模式的优势在于处理复杂信息，信息越完整，分析越好
3. **关注思考过程**——展开"Thinking"区域阅读 Claude 的推理链，你可以从中学到分析问题的方法论
4. **结合 Projects 使用**——在项目中启用深度思考，Claude 会基于知识库做更深入的分析

还有一个进阶用法：当你对 Claude 的某个分析结论有疑问时，可以要求它"重新思考这个问题，特别关注之前可能忽略的反面论据"。这样可以触发更全面的第二轮深度推理，帮你发现盲点。
