# 模块2：学会提问

## 本节要点

- 为什么问法决定了AI回答的质量
- Prompt 的4个构成要素
- 10个真实工作场景示范
- 如何迭代优化你的 Prompt

## 4要素框架回顾

| 要素 | 说明 | 例子 |
|------|------|------|
| 角色 | AI 扮演什么专家 | 「你是资深营销策划师」 |
| 任务 | 你想要什么 | 「帮我写一份活动方案」 |
| 背景 | 相关的上下文 | 「预算5万，目标用户是95后」 |
| 格式 | 输出什么格式 | 「用PPT大纲的格式」 |

## 课后练习

用4要素框架，把下面这个模糊需求改写成好 Prompt：

> "帮我想想这个产品怎么推广"
