# 模块1：认识Claude

## 本节要点

- Claude 是什么（不是什么）
- Claude 最擅长的5类任务
- 免费版够用吗？套餐对比
- 完成你的第一次对话

## 课后练习

打开 claude.ai，完成以下3个对话：

1. 让 Claude 自我介绍（不超过100字）
2. 问它「今天上海天气怎么样」（观察它怎么处理不知道答案的情况）
3. 让它帮你写一封请假条

把你的体验分享到评论区！
